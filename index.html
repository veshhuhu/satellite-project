🚀 Satellite Ground Track Project

This website visualizes a satellite ground track based on orbital elements derived from a state vector.

🔢 Orbital Period

The calculated orbital period is approximately 14.94 hours using Kepler's Third Law.

📘 Question 1: Orbital Elements and Path Calculation

The satellite's orbit was calculated using orbital elements (a, e, i, RAAN, ω, M₀). These were propagated using Kepler’s equation and transformed into lat/lon considering Earth rotation.

🌍 Question 2: Ground Track Map

The satellite’s path was visualized over a world map with grid, marker, and terminator line. An animated GIF shows the full orbit.
